# Get Started

This guide will help you get started with OpenSlop in no time.

## Installation

Run the following command in your terminal to install OpenSlop:

```bash
npm i -g @openslop/openslop
```

## Quick Start

After installation, you can setup the OpenSlop by running the following command:
You will be prompted to enter the [baseUrl](/glossary#base-url), [apiKey](/glossary#api-key) and [model name](/glossary#model-name).

```bash
openslop setup
```

Next, check out the [Configuration](/config) guide to customize it for your needs.
