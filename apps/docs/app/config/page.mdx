# Configuration

The `.openslop/config.json` file is the base of OpenSlop. It stores [baseUrl](/glossary#base-url), [apiKey](/glossary#api-key) and [model name](/glossary#model-name) for the agent.

## Basic Options

Below is the schema of the config file. You can find it in your user folder under .openslop

```json
{
  baseUrl: "string",
  apiKey: "string",
  model: "string"
}
```

If you haven't yet, read about [Getting Started](/get-started).
