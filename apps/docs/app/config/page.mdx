# Configuration

OpenSlop can be customized easily through the `.openslop/config.json` file.

## Basic Options

Here are some of the basic configuration options you can provide:

```json
{
  baseUrl: "provider-base-url",
  apiKey: "your-api-key",
  model: "llm-model-name"
}
```

If you haven't yet, read about [Getting Started](/get-started).
